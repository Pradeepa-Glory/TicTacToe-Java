# Tic Tac Toe Game (Java)

A command-line based Tic Tac Toe game developed in Java.  
This project allows two players to play against each other on a 3x3 grid.

---

## Features
- Two-player mode
- Player name input
- Turn-based gameplay (X and O)
- Input validation
- Win and draw detection
- Console-based UI with grid format

---

## How to Play
1. Enter the names of both players.
2. Player 1 plays with **X**, Player 2 plays with **O**.
3. Enter row and column numbers (0 to 2).
4. The first player to align three symbols horizontally, vertically, or diagonally wins.

---
